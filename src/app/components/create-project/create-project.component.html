Initialiser un repository DockerHub

Ton code source doit être intégralement sur Github ! Il faut ensuite que tu créés un fichier nommé Dockerfile à la racine de ton projet avec le contenu suivant :

{{ dockerfile }}

Tu dois te créer un compte sur DockerHub si tu n'en as pas.
